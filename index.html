Proyectos

OddStats

OddStats es un webscrapper que recolecta datos de partidos de una casa de apuestas. Este bot esta alojado en un servidor que mediante Python y la librería BeautifulSoup va comprobando las distintas cuotas cada cierto tiempo para después almacenarlas en una base de datos MySql

El bot lleva en marcha desde septiembre de 2020 y actualmente (Enero 2021) lleva recolectados 25.000 partidos de 24 deportes. En total lleva detectadas alrededor de 19 Millones de cuotas.

Mediante esta base de datos he creado varias plataformas para acceder a los datos y así aprender distintos lenguajes y frameworks.

  • Python Dashboard

Con la ayuda de Python y la librería de Dash he creado un panel interactivo para acceder de una manera rápida a los datos con una visualización detallada con graficos de diferentes tipos. Estos graficos han sido creados con la librería

La finalidad de este proyecto es totalmente educativo y no es usado para fines comerciales. Sin animo de lucro
