OddStats

OddStats es un webscrapper que recolecta datos de partidos de una casa de apuestas. Este bot esta alojado en un servidor que mediante Python y la librería BeautifulSoup va comprobando las distintas cuotas cada cierto tiempo para después almacenarlas en una base de datos MySql

El bot ha estado funcionando las 24/7 desde septiembre de 2020 hasta enero de 2021. En este tiempo a analizado 31,500 partidos y recolectado 23 Millones de cuotas

Mediante esta base de datos he creado varias plataformas para acceder a los datos y así aprender distintos lenguajes y frameworks.

  • Api de Oddstats: kirostats_api.aimarperuchena.com/
  • Página web Dash con Python: kirostats_dash.aimarperuchena.com Login: aimar/aimar

  • Python Dashboard

Con la ayuda de Python y la librería de Dash he creado un panel interactivo para acceder de una manera rápida a los datos con una visualización detallada con graficos de diferentes tipos. Estos graficos han sido creados con la librería Plotly, y para dar estilos a la pagina web Bootstrap .

  • Kirolstats Laravel

Con la ayuda de Php y la librería Laravel he creado una pagina web con php para poner en practica mis conocimientos en php. Tiene activada la autentificación pero esta en proceso de terminar la parte autenticada. Finalmente para dar estilos a la pagina web Bootstrap .

La finalidad de este proyecto es totalmente educativa y fines comerciales. Sin animo de lucro
